# 👻 Ghost-Trace – Gesture Based Interaction System

**Ghost-Trace** is a browser-based interactive project that uses **hand tracking technology** to detect user gestures and trigger real-time visual effects.

Inspired by anime-style energy effects, this project creates an immersive experience using live webcam input and gesture recognition.

---

## 📌 Overview

Ghost-Trace captures webcam input and tracks hand movements using **MediaPipe Hands**. Based on gesture detection (like open/closed hands), it dynamically triggers animated visual effects on screen.

It demonstrates real-time computer vision directly in the browser.

---

## 🎯 Features

- ✋ Real-time hand tracking  
- 🎥 Webcam-based interaction  
- 🎨 Dynamic visual effects  
- 🔥 Gesture-triggered animations  
- 🪞 Mirror camera view  
- ⚡ Smooth rendering with Canvas

## ⚙️ How It Works

1. Webcam is accessed via browser  
2. MediaPipe detects hand landmarks  
3. System checks gesture state (open/closed)  
4. Based on gestures:
   - Effects are triggered  
5. Animations appear on screen
